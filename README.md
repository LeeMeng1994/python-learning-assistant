# Python 学习助手

一个基于 PyQt6 的 Python 入门学习应用，内置 10 节互动课程，帮助零基础用户快速掌握 Python 编程。

## 功能特性

- **10 节内置课程**：从 Hello World 到文件操作，循序渐进
- **互动练习**：每节课配套练习题，即时反馈答案
- **代码高亮**：内置 Python 语法高亮编辑器
- **成就系统**：完成课程解锁成就，增加学习动力
- **进度保存**：自动记录学习进度，下次继续

## 课程目录

1. Hello World - 第一个 Python 程序
2. 变量与数据类型
3. 条件判断 if
4. for 循环
5. while 循环
6. 函数 def
7. 列表 list
8. 字典 dict
9. 文件操作
10. 异常处理

## 运行方式

```bash
pip install PyQt6
python main.py
```

## 打包

```bash
pip install pyinstaller
pyinstaller Python学习助手.spec
```

## 技术栈

- Python 3.12
- PyQt6
- PyInstaller
